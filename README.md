# 学生宿舍管理系统 python651

## 项目概述

学生宿舍管理系统是一款使用Python语言，结合SQL Server/MySQL数据库以及HTML技术开发的软件。系统功能全面，旨在简化宿舍管理工作流程，提高管理效率。

## 技术栈

- **后端语言**：Python
- **数据库**：SQL Server / MySQL
- **前端技术**：HTML

## 功能模块

- **学生信息管理**
  - 学生信息录入
  - 学生信息查询
  - 学生信息修改
  - 学生信息删除
- **学生宿舍管理**
  - 宿舍分配查询
- **管理员功能**
  - 管理员登录
  - 全局数据查询
  - 数据修改与删除

## 系统角色

- **学生**：查询个人信息与宿舍分配情况
- **管理员**：管理系统数据，包括录入、查询、修改与删除

## 运行环境

- Python运行环境
- SQL Server或MySQL数据库环境
- 支持HTML的Web浏览器

## 部署步骤

1. 环境配置：安装Python环境、SQL Server/MySQL数据库
2. 数据库搭建：根据项目需求创建数据库及数据表
3. 代码部署：配置后端Python代码与前端HTML页面
4. 测试运行：进行系统功能测试，确保无误
